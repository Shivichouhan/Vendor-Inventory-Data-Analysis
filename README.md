# Vendor Performance & Inventory Data Analysis

## Project Overview
This project analyzes retail inventory data to help a business understand its stock levels, vendor spending, and product performance. I used **Python** and **SQL** to transform raw CSV data into actionable business insights.

## Key Technical Skills
* **Data Manipulation:** Python (Pandas)
* **Database Management:** SQL (SQLite & SQLAlchemy)
* **Data Visualization:** Matplotlib & Seaborn
* **Environment:** Google Colab

## Business Insights Discovered
1. **Top Vendors:** Identified the highest spending vendors to help in price negotiations.
2. **Inventory Value:** Calculated the total capital locked in the warehouse.
3. **Product Turnover:** Found which brands are moving fast and which are sitting idle.

## How to Run
1. Upload the CSV files to a Google Colab environment.
2. Run the provided `.ipynb` notebook.
